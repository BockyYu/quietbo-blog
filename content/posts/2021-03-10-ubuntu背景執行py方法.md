主要介紹在linux下使用Terminal來背景執行python的幾種方式，  
執行3隻python檔，每5秒、10秒、15秒各別顯示當前時間。

以下為python範例:  
test.py、test2.py、test3.py這三隻同時使用背景來執行的畫面。

執行test.py後，會顯示跳出PID為25270。

<pre class="wp-block-code"><code lang="python" class="language-python line-numbers"># coding:utf-8
from datetime import datetime
from time import sleep

while(1):
    count = 1
    sleep(5)
    print(str(count) + " : " + str(datetime.now().strftime('%Y-%m-%d %H:%M:%S')))</code></pre>

如果想讓程式跑在背景，在結尾的地方加個&  
python 背景執行指令如下:
