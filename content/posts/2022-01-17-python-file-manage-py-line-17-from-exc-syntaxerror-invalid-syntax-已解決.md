我在env內，有python2.7與python3.8。  
當使用 python 運行命令時，Django 無法預測確切的 python 版本。

執行下列指令後出現錯誤:

<pre class="wp-block-code"><code class="">python manage.py runserver</code></pre>

問題訊息如下:

<pre class="wp-block-code"><code class="">  File "manage.py", line 17
    ) from exc
         ^
SyntaxError: invalid syntax</code></pre><figure class="wp-block-image">

<img decoding="async" src="https://i.imgur.com/nzdZu0Z.png" alt="" /> </figure> 

解決方式:

<pre class="wp-block-code"><code class="">python3 manage.py runserver
python3 manage.py runserver 0.0.0.0:8899 # 轉port為8899</code></pre>
