## <span class="ez-toc-section" id="%E6%95%B4%E6%95%B8int"></span>整數int<span class="ez-toc-section-end"></span> {.wp-block-heading}

[microsoft(整數的數字型別)][1]

<pre class="wp-block-code"><code lang="csharp" class="language-csharp line-numbers">int apple = 100;  // 宣告變數apple時就給值100

int bird; // 宣告變數bird，但不給值
bird = 200; // 設定變數bird的值為200

int lucky_girl_age = 18, lucky_boy_age = 17, abby; // 宣告數個同型別的變數</code></pre>

## <span class="ez-toc-section" id="var"></span>var<span class="ez-toc-section-end"></span> {.wp-block-heading}

<ul class="wp-block-list">
  <li>
    必需宣告值<br />var是一定要先給值，var是讓電腦來區分值是什麼型別，若沒有給值，電腦會無法判斷。
  </li>
</ul>

在編譯時會經由10,推測number的型別是整數:

<pre class="wp-block-code"><code lang="csharp" class="language-csharp line-numbers">var number = 10; </code></pre>

不能只宣告的:var

## <span class="ez-toc-section" id="%E6%B5%AE%E9%BB%9E%E6%95%B8float%E3%80%81double%E3%80%81decimal"></span>浮點數(float、double、decimal)<span class="ez-toc-section-end"></span> {.wp-block-heading}

[microsoft(浮點數值型別)][2]  
每個浮點數型別的預設值都是零，浮點數型別都有 MinValue 與 MaxValue 常數，提供該型別的最小與最大有限值。

<ol class="wp-block-list">
  <li>
    float
  </li>
</ol>

<ul class="wp-block-list">
  <li>
    單精準數
  </li>
  <li>
    不可以直接只指定數值，必需在數值最後加 F 或 f
  </li>
</ul>

<ol class="wp-block-list">
  <li>
    double
  </li>
</ol>

<ul class="wp-block-list">
  <li>
    雙精準數
  </li>
  <li>
    可以直接指定數值，或在數值最後加 D 或 d
  </li>
</ul>

<ol class="wp-block-list">
  <li>
    decimal
  </li>
</ol>

<ul class="wp-block-list">
  <li>
    進位數
  </li>
  <li>
    不可以直接只指定數值，必需在數值最後加M或m
  </li>
  <li>
    通常用於財務(有關錢的計算)。
  </li>
</ul>

從c# 7.0開始支援_做為數位分隔符號，可以使用數位分隔符號搭配所有種類的數值常值。

<pre class="wp-block-code"><code lang="csharp" class="language-csharp line-numbers">double d_1 = 9D;
double d_2 = 6d;
double d_3 = 4.852_0001000;
Console.WriteLine(d_3); // 4.8520001

float f_1 = 2_000.5F;
float f_2 = 3.4000f;
Console.WriteLine(f_2); // 3.4

decimal myMoney_1 = 3_000.50m;
decimal myMoney_2 = 400.750000M;
Console.WriteLine(myMoney_2); // 400.750000</code></pre>

也可以使用科學記號標記法。

<pre class="wp-block-code"><code lang="csharp" class="language-csharp line-numbers">double d = 0.42e2;
Console.WriteLine(d);  // 42

float f = 134.45E-2f;
Console.WriteLine(f);  // 1.3445

decimal m = 1.5E6m;
Console.WriteLine(m);  // 1500000</code></pre>

## <span class="ez-toc-section" id="%E9%99%A3%E5%88%97arry"></span>陣列(arry)<span class="ez-toc-section-end"></span> {.wp-block-heading}

<ul class="wp-block-list">
  <li>
    指定長度後就不可以再修改長度，較適合已知長度
  </li>
  <li>
    第一筆資料從0開始
  </li>
  <li>
    型別必須一致
  </li>
  <li>
    可以重覆修改值
  </li>
</ul>

格式:

<pre class="wp-block-code"><code class="">資料型別 [] 陣列名稱 = new 資料型別[陣列大小];</code></pre>

<pre class="wp-block-code"><code lang="csharp" class="language-csharp line-numbers">int[] numbers = new int[4]; // 陣列長度為4
// 宣告後再把值填到陣列裡
numbers[0] = 1000;
numbers[1] = 10;
numbers[2] = 20;
numbers[3] = 30;
numbers[2] = -100; //修改值 </code></pre>

宣告陣列時填入值：

<pre class="wp-block-code"><code lang="csharp" class="language-csharp line-numbers">// 雖然一開始沒指定長度，但長度是固定的
string[] items = new[] {"A", "B", "C", "D", "E"};
// items[5] = "Z" //會報錯，陣列的長度已超過
/*
 * 長度只到5，會出現錯誤的訊息：
 * System.IndexOutOfRangeException: Index was outside the bounds of the array.
*/</code></pre>

取得陣列長度:

<pre class="wp-block-code"><code lang="csharp" class="language-csharp line-numbers">string[] items = new[] {"A", "B", "C", "D", "E"};
int length = items.Length;</code></pre>

排序小到大＆大到小：

<pre class="wp-block-code"><code lang="csharp" class="language-csharp line-numbers">int[] items = new int[] {8, 10, 1, 3, 5};
Array.Sort(items); // 小到大
//Array.Reverse(items); // 轉成大到小需先使用sort排列過
foreach (var item in items)
{
    Console.Write($"{item}\t");
}
// 1    3   5   8   10</code></pre>

Sort(小到大）允許傳入兩個陣列;  
下方使用2個不同的陣列來做成績的排序，且名字要與分數一致。

<pre class="wp-block-code"><code lang="csharp" class="language-csharp line-numbers">string[] students = {"小王", "小美", "大強", "大白", "自強", "莒光"};
int[] score_math = {80, 50, 90, 66, 76, 99};

// 將score_math排序, 且同時用相同順序排列students
Array.Sort(score_math, students);

//若想要遞增，則需使用2次Reverse
// Array.Reverse(score_math);
// Array.Reverse(students);
for (int i = 0; i&lt; students.Length; i++)
{
    string name = students[i];
    int score = score_math[i];
    Console.WriteLine($"{name}的成績:{score}");
}</code></pre>
