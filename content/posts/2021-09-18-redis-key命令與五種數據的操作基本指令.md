## <span class="ez-toc-section" id="Key%E5%91%BD%E4%BB%A4"></span>Key命令<span class="ez-toc-section-end"></span> 



<ul class="wp-block-list">
  <li>
    在查找key的時候支持正則
  </li>
</ul>

<span style="font-size: revert; color: initial;">格式</span>:

<pre class="wp-block-code"><code class="">keys pattern</code></pre>

範例:

<pre class="wp-block-code"><code class="">keys *    # 查所有key，常用
keys 'k*' # 查名稱包含[開頭為k]的key</code></pre><figure class="wp-block-image">

<img decoding="async" src="https://i.imgur.com/dGtaC1H.png" alt="" /> </figure> 

### <span class="ez-toc-section" id="%E6%AA%A2%E6%9F%A5Key%E6%98%AF%E5%90%A6%E5%AD%98%E5%9C%A8"></span>檢查Key是否存在<span class="ez-toc-section-end"></span> 

格式(可多個): 存在返回1，不存在返回0

<pre class="wp-block-code"><code class="">exists key1</code></pre>

範例:

<pre class="wp-block-code"><code class="">exists k0
exists k1
exists k0 k1
exists k0 k1 k2 k3</code></pre><figure class="wp-block-image">

<img decoding="async" src="https://i.imgur.com/EYs4V02.png" alt="" /> </figure> 

### <span class="ez-toc-section" id="%E6%9F%A5%E7%9C%8BKey%E5%B0%8D%E6%87%89%E7%9A%84value%E7%9A%84%E9%A1%9E%E5%9E%8BType"></span>查看Key對應的value的類型(Type)<span class="ez-toc-section-end"></span> 

格式:

<pre class="wp-block-code"><code class="">type key</code></pre>

redis⽀持的五種類型:

<ol class="wp-block-list">
  <li>
    string
  </li>
  <li>
    list
  </li>
  <li>
    set
  </li>
  <li>
    zset
  </li>
  <li>
    hash
  </li>
</ol><figure class="wp-block-image">

<img decoding="async" src="https://i.imgur.com/r0GYnew.png" alt="" /> </figure> 

### <span class="ez-toc-section" id="%E5%88%AA%E9%99%A4Key"></span>刪除Key<span class="ez-toc-section-end"></span> 

格式:

<pre class="wp-block-code"><code class="">del key1
del key1 key2 key3 # 刪除多個</code></pre><figure class="wp-block-image">

<img decoding="async" src="https://i.imgur.com/7qjMFEG.png" alt="" /> </figure> 

### <span class="ez-toc-section" id="%E8%A8%AD%E7%BD%AEKey%E9%81%8E%E6%9C%9F%E6%99%82%E9%96%93"></span>設置Key過期時間<span class="ez-toc-section-end"></span> 

如果在一開始就沒有指定過期時間，Key會一直存在，直到用Del才會移除，  
在已有的Key設置過期時間(以秒為單位):

    刪除指定的值
  </li>
  <li>
  </li>

<pre class="wp-block-code"><code class="">lrem key count value</code></pre>

移除元素可以是一個或多個，都是照count來決定移除的數量和方向。

<ol class="wp-block-list">
  <li>
    count > 0 : 從元素0開始，移除count個與value相同的元素
  </li>
  <li>
    count < 0 : 從最後一個元素，移除count個與value相同的元素
  </li>
  <li>
    count = 0 : 移除表中與value相同的元素
  </li>
</ol>

範例:

<pre class="wp-block-code"><code class="">rpush lrem_test aa bb cc oo aa zz bb aa rr aa  # 先建立一個list
lrange lrem_test 0 -1  # 檢視一下新創立的list
lrem lrem_test 2 aa  # 刪除2個與aa相同的元素 (下方圖一)
lrem lrem_test -1 aa  # 移除表中後面數的第1個與aa相同的元素 (下方圖二)
lrem lrem_test 0 bb  # 移除表中與bb相同的元素 (下方圖三)</code></pre>

## <span class="ez-toc-section" id="zset%E6%9C%89%E5%BA%8F%E9%9B%86%E5%90%88"></span>zset有序集合<span class="ez-toc-section-end"></span>
