符號:@  
用途:加在字串前面，表示其中的轉義字元不被處理，也可換行。

<pre class="wp-block-code"><code lang="csharp" class="language-csharp line-numbers">string a = "hello \t world";
string b = @"hello \t world";
string c = "today is \"Sunday\"";
string d = @"today is ""Sunday""";
string e = "https:\\\\localhost\\user\\name.txt";
string f = @"https:\\localhost\user\name.txt";
string g = "AAA\r\nBBB\r\nCCC";
string h = @"
AAAA
BBBB
CCCC";</code></pre>

顯示如下:  
<img decoding="async" src="https://i.imgur.com/6u5UR8t.png" alt="" /> 

<blockquote class="wp-block-quote is-layout-flow wp-block-quote-is-layout-flow">
  <p>
    如果使用@的話，編譯時會自動上\r\n\t。像h輸出就是自動加換行的結果。<br />此處注意的是:顯示一個雙引號的話，需要像d寫法一樣，使用2個雙引號&#8221;&#8221;才會當作1個雙引號&#8221;
  </p>
</blockquote>

符號:$  
用途:使用String.Format將指定的變數名稱帶入字串中。  
其實就是:字串輸出格式。

<pre class="wp-block-code"><code lang="csharp" class="language-csharp line-numbers">string a = "hello world";
string b = @"today is ""Sunday""";
string c = @"https:\\localhost\user\name.txt";


string i = $"{a} {b} URL:{c}";
Console.WriteLine(i);</code></pre>

輸出:

<pre class="wp-block-code"><code class="">hello world today is "Sunday" URL:https:\\localhost\user\name.txt</code></pre>
