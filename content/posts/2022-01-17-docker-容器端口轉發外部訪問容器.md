容器中可以運行一些網絡應用，要讓外部也可以訪問這些應用，可以通過 -P 或 -p 參數來指定端口映射。  
當使用 -P 標記時，Docker 會隨機映射一個端口到內部容器開放的網絡端口。



使用:Mac M1。

創建容器，注意port已將80轉出到8888

<pre class="wp-block-code"><code lang="bash" class="language-bash">docker container run -d --rm --name web -p 8888:80 nginx</code></pre>

查看IP:

<pre class="wp-block-code"><code lang="bash" class="language-bash">$ docker inspect --format '{{ .NetworkSettings.IPAddress }}' web
172.17.0.3</code></pre>

創建image busybox，名為client的容器:

<pre class="wp-block-code"><code lang="bash" class="language-bash">docker run -d --rm --name client busybox /bin/sh -c "while true; do sleep 3600; done"</code></pre>

下載web的index.html:

<pre class="wp-block-code"><code class="">$ docker exec -it client wget http://172.17.0.3
Connecting to 172.17.0.3 (172.17.0.3:80)
saving to 'index.html'
index.html           100% |********************************|   615  0:00:00 ETA
'index.html' saved</code></pre>

輸入:ifconfig，查找自己的IP，如下:192.168.1.108

<pre class="wp-block-code"><code class="">en0: flags=8863&lt;UP,BROADCAST,SMART,RUNNING,SIMPLEX,MULTICAST&gt; mtu 1500
    options=6463&lt;RXCSUM,TXCSUM,TSO4,TSO6,CHANNEL_IO,PARTIAL_CSUM,ZEROINVERT_CSUM&gt;
    ether a0:78:17:a6:e9:f5
    inet6 fe80::c75:fc71:85f1:6b48%en0 prefixlen 64 secured scopeid 0xb

    nd6 options=201&lt;PERFORMNUD,DAD&gt;

    status: active</code></pre>

開啟瀏覽器輸入:

<pre class="wp-block-code"><code lang="bash" class="language-bash">192.168.1.108:8888</code></pre>
