之前在網路上找了一下，很多都要去git clone 字體才能解決，  
但最簡單的方式就是直接輸入。

以下是用Mac操作如下：  
code -> 喜好設定 -> 設定 -> 輸入Font Family  
<img decoding="async" src="https://i.imgur.com/a8w62Do.png" alt="" /> 

終端機 -> Integrated:Font Family內，輸入&#8217;Meslo LG L DZ for Powerline&#8217;  
<img decoding="async" src="https://i.imgur.com/bOjV4pf.png" alt="" /> 

完成！



如果是在Windows的話，可能沒有 Meslo LG L DZ for Powerline 這個字型，可以參考這篇<a href="https://quietbo.com/2021/08/18/windows-%e7%be%8e%e5%8c%96terminal-powershell/" data-type="URL" data-id="https://quietbo.com/2021/08/18/windows-%e7%be%8e%e5%8c%96terminal-powershell/">[Windows] 美化Terminal- powershell</a>，主要是字型有Source Code Pro for Powerline，就可以在Source Code Pro for Powerline輸入上，也可正常使用。
