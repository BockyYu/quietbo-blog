## <span class="ez-toc-section" id="%E5%88%AA%E9%99%A4%E5%AD%97%E4%B8%B2%E5%B7%A6%E9%82%8A_%E5%8F%B3%E9%82%8A_%E5%85%A9%E5%81%B4%E7%9A%84%E7%A9%BA%E7%99%BD"></span>刪除字串左邊, 右邊, 兩側的空白<span class="ez-toc-section-end"></span> 

<ul class="wp-block-list">
  <li>
    TrimStart() 刪除左空白
  </li>
  <li>
    rimEnd() 刪除右空白
  </li>
  <li>
    Trim() 刪除左右空白
  </li>
</ul>

範例：

<pre class="wp-block-code"><code lang="csharp" class="language-csharp line-numbers">string str_name = "  bocky Bocky ";
string trim_start = str_name.TrimStart();
string trim_end = str_name.TrimEnd();
string trim_reim = str_name.Trim();</code></pre>

結果如下：

<pre class="wp-block-code"><code lang="csharp" class="language-csharp line-numbers">bocky Bocky 
  bocky Bocky
bocky Bocky</code></pre>

## <span class="ez-toc-section" id="%E7%8D%B2%E5%BE%97%E5%AD%97%E4%B8%B2%E7%9A%84%E9%95%B7%E5%BA%A6"></span>獲得字串的長度<span class="ez-toc-section-end"></span> 

最常使用在獲得帳號或密碼的長度上，做長度的限制。

<pre class="wp-block-code"><code lang="csharp" class="language-csharp line-numbers">string password = "iambocky";
int password_len = password.Length;
Console.WriteLine("密碼長度:" + password_len); // 8</code></pre>

## <span class="ez-toc-section" id="%E6%88%AA%E5%8F%96%E5%AD%97%E4%B8%B2"></span>截取字串<span class="ez-toc-section-end"></span> 

<ul class="wp-block-list">
  <li>
    擷取的位置需在字串範圍
  </li>
</ul>

<pre title="" class="wp-block-code"><code lang="csharp" class="language-csharp line-numbers">string name = "iambocky";
string news_name = name.Substring(3, 5); 
Console.WriteLine(news_name);</code></pre>

## <span class="ez-toc-section" id="%E5%AD%97%E4%B8%B2%E5%87%BA%E7%8F%BE%E6%9F%90%E4%BA%9B%E5%AD%97%E7%9A%84%E4%BD%8D%E7%BD%AE"></span>字串出現某些字的位置<span class="ez-toc-section-end"></span> 

<ul class="wp-block-list">
  <li>
    IndexOf() 由左至右開始檢查，傳回第一個符合的位置,若字串中沒有符合則傳回-1
  </li>
  <li>
    LastIndexOf() 是從字串最後（右向左）尋找, 傳回第一個符合的位置
  </li>
</ul>

範例：判斷字串裡是否有@

<pre class="wp-block-code"><code lang="csharp" class="language-csharp line-numbers">string email = "bocky@gmail.com";
int location = email.IndexOf('@');
Console.WriteLine(location); // @所在的位置（從0開始）
if (location &gt; 0)
{
    Console.WriteLine("屬於email");
}
else
{
    Console.WriteLine("不屬於email");
}</code></pre>

## <span class="ez-toc-section" id="%E5%AD%97%E4%B8%B2%E5%8F%96%E4%BB%A3"></span>字串取代<span class="ez-toc-section-end"></span> 

將字串裡的某一個字,或某個字串, 置換成其他字。

不想要空白，就把空白取代為，範例：

<pre class="wp-block-code"><code lang="csharp" class="language-csharp line-numbers">string value = " te st .jos n ";
string result = value.Replace(" ", "");
Console.WriteLine(result); // test.josn</code></pre>

## <span class="ez-toc-section" id="%E5%AD%97%E4%B8%B2%E5%88%87%E5%89%B2"></span>字串切割<span class="ez-toc-section-end"></span> 

將一個字串分割成數個子字串。  
Split()

舉例要取年月日，用/來做切割，切割出來的資料需存在陣列內。

<pre class="wp-block-code"><code lang="csharp" class="language-csharp line-numbers">string today = "2021/10/31";
string[] tmp = today.Split('/');

string year = tmp[0];
string month = tmp[1];
string day = tmp[2];

Console.WriteLine(year); // 2021
Console.WriteLine(month); // 10
Console.WriteLine(day); //31</code></pre>

## <span class="ez-toc-section" id="%E5%AD%97%E4%B8%B2%E8%BD%89%E6%8F%9B"></span>字串轉換<span class="ez-toc-section-end"></span> 

<ul class="wp-block-list">
  <li>
    ToUpper() 全部大寫
  </li>
  <li>
    ToLower() 全部小寫
  </li>
</ul>

大部分需要全部轉換成大小寫一致的話是為了比對。

<pre class="wp-block-code"><code lang="csharp" class="language-csharp line-numbers">string name = "Bocky";
Console.WriteLine(name.ToUpper()); //全部大寫BOCKY
Console.WriteLine(name.ToLower()); //全部小寫bocky</code></pre>

補充:

<pre class="wp-block-code"><code class="">若想判斷二個變數內的字串文字是否相同, 建議使用：Equals() </code></pre>
