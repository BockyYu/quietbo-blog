# <span class="ez-toc-section" id="Git_%E5%9F%BA%E6%9C%AC%E8%A8%AD%E5%AE%9A%E4%BB%A5%E5%8F%8APush_Hearoku"></span>Git 基本設定以及Push Hearoku<span class="ez-toc-section-end"></span> 

查看目前的設定

<pre class="wp-block-code"><code lang="bash" class="language-bash">git config --list</code></pre>

如果本來就沒有設定git，需要輸入以下指令來進行初始化:

<pre class="wp-block-code"><code lang="bash" class="language-bash">git config –global user.name “你的名字”
git config –global user.email “你的信箱”</code></pre>

在自己的專案創建一個.git檔案(該動作只需要做一次)

<pre class="wp-block-code"><code lang="bash" class="language-bash">git init </code></pre>

回到資料夾檢查看有沒有出現.git的資料夾，  
通常是被隱藏起來，想看的話要把檢視->隱藏的項目打勾  
<img decoding="async" src="https://i.imgur.com/qafdlGn.png" alt="" />  
<img decoding="async" src="https://i.imgur.com/P8iE5aM.png" alt="" /> 

將 git 資料夾與 Heroku作為連接，這邊HEROKU\_APP\_NAME是要照你當時在Heroku的專案名稱一樣，  
如果不同，他則會報錯

<pre class="wp-block-code"><code lang="bash" class="language-bash">heroku git:remote -a {HEROKU_APP_NAME}</code></pre>

下圖為錯誤訊息:  
<img decoding="async" src="https://i.imgur.com/aomEjdG.png" alt="" />  
下圖為設定成功的訊息:  
<img decoding="async" src="https://i.imgur.com/mwlCRRR.png" alt="" /> 

然後照下圖方式  
<img decoding="async" src="https://i.imgur.com/axvyAxT.png" alt="" />  
指令如下:

<pre class="wp-block-code"><code lang="bash" class="language-bash">git add .
git commit -m "你想輸入的訊息(必填)" 
git push -f heroku master</code></pre>

最後會執行一段時間，成功如下圖  
<img decoding="async" src="https://i.imgur.com/ndZrJNC.png" alt="" /> 

# <span class="ez-toc-section" id="Line_Bot_%E8%88%87_Hearoku_%E7%B6%81%E5%AE%9A"></span>Line Bot 與 Hearoku 綁定<span class="ez-toc-section-end"></span> 

剛剛已經完成將程式碼PUSH到Hearoku了，現在則是讓Line Bot能夠到Hearoku去運行我們想要機器人為我們做的事情。

回到LINE Developers的Messaging API，  
選擇 Webhook URL 點擊Edit 將網址貼上去，格式:

<pre class="wp-block-code"><code lang="bash" class="language-bash">https://{HEROKU_APP_NAME}.herokuapp.com/callback</code></pre>

輸入完後點選Verify，成功後就會將剛剛輸入的URL存上去了。  
<img decoding="async" src="https://i.imgur.com/oWzuUvB.png" alt="" /> 

回到Heroku網站，點選Open app，跳出的新網頁就是你的專案網址。

測試自己做的機器人回覆內容  
<img decoding="async" src="https://i.imgur.com/8Qh5Qm8.png" alt="" /> 

目前機器人是看打什麼字就回復什麼內容，  
如果要關閉每次都跳出那感謝訊息，要到回到回應設定，將自動回覆訊息關閉。  
<img decoding="async" src="https://i.imgur.com/JNwMkAj.png" alt="" /> 

# <span class="ez-toc-section" id="clone_heroku_%E7%9A%84%E7%A8%8B%E5%BC%8F%E7%A2%BC"></span>clone heroku 的程式碼<span class="ez-toc-section-end"></span> 

有些人開發不一定是使用同一台電腦，clone的指令如下:

<pre class="wp-block-code"><code lang="bash" class="language-bash">git clone https://git.heroku.com/YOUR_HEROKU_APP_NAME.git</code></pre>

在`YOUR_HEROKU_APP_NAME`這串英文改成你的專案名字就可以成功clone下來了。
