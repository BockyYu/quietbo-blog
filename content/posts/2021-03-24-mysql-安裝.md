apt套件管理器安裝:

<pre class="wp-block-code"><code lang="bash" class="language-bash">sudo apt update
sudo apt install mysql-server -y</code></pre>

安裝完應會自動啟動:  
<img decoding="async" src="https://i.imgur.com/SE8x1aZ.png" alt="" /> 

檢查redis服務狀態:

<pre class="wp-block-code"><code lang="bash" class="language-bash">service mysql status</code></pre>

停止redis server:

<pre class="wp-block-code"><code lang="bash" class="language-bash">service mysql stop</code></pre>

啟動redis server:

<pre class="wp-block-code"><code lang="bash" class="language-bash">service mysql start</code></pre>

進入Mysql:

<pre class="wp-block-code"><code lang="bash" class="language-bash">mysql -u root -p</code></pre><figure class="wp-block-image">

<img decoding="async" src="https://i.imgur.com/WeiyuQw.png" alt="" /> </figure> 

若出現則登入成功。
