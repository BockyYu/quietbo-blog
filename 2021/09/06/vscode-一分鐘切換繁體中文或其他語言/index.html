[VSCode] 一分鐘切換繁體中文或其他語言

Monday, September 6, 2021 1 minute read
  1. 安裝中文套件chinese

  1. 顯示命令面板Show Command Palette
    windows 按ctrl + shift + P 或 F1
    Mac 按 cmd + shift + P
  2. 輸入 display language
    選擇跳出的 Configure Display Language
  3. 選擇想要的語言

重新啟動VSCode即完成

如果要選擇其他語言，就回到1去找尋該語言的套件，安裝後再執行2.3.4的步驟
