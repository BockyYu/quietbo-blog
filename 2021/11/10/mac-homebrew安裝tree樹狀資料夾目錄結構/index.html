[Mac] Homebrew安裝tree(樹狀資料夾目錄結構)

Tuesday, November 9, 2021 1 minute read

目的：使用homebrew來安裝tree，並顯示資料結構。

Homebrew

可以在 Mac 上安裝系統沒有的套件，例如 Wget (算是工程師滿常使用的軟體)

開啟終端機，輸入下方指令來安裝Homebrew

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

更新套件

brew update && brew upgrade && brew cleanup

安裝tree

brew install tree

若在顯示列印資料夾目錄，中文檔名有亂碼時，加上 -N

tree -N

有些資料夾層級太多時，可指定要印出的層數，後面加上層數的數字：

tree -L 1

依字母排序

tree -v

關鍵字搜尋 Homebrew 套件(下方是搜尋tree)

brew search tree

查看更多tree的功能：

tree --help
