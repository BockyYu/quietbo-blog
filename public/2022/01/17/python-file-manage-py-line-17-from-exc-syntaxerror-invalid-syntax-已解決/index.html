[Python] File “manage.py”, line 17 ) from exc ^ SyntaxError: invalid syntax (已解決)

Monday, January 17, 2022 1 minute read

我在env內，有python2.7與python3.8。
當使用 python 運行命令時，Django 無法預測確切的 python 版本。

執行下列指令後出現錯誤:

python manage.py runserver

問題訊息如下:

  File "manage.py", line 17
    ) from exc
         ^
SyntaxError: invalid syntax

解決方式:

python3 manage.py runserver
python3 manage.py runserver 0.0.0.0:8899 # 轉port為8899
