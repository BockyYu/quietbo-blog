[解決]終端機的亂碼’?’

Thursday, May 6, 2021 1 minute read

最近買了一台mac，網路上找了安裝iTerms2 + Oh-my-zsh的方法，順利完成安裝後，再回頭開原本的Terminal(終端機)，發現竟然有亂碼！？
雖然絕大部分都開iTerms2，但還是想把這問題解決，不然我晚上一定睡不著覺。
本篇不是要解決終端機中文字出現亂碼問題。

下圖是終端機的樣子，有“？”

下圖為iTerm2

為什麼原本的終端機會有？但iTerms2沒有呢？

主要是因為一個叫Powerline的字型，如果有興趣的人可以解決這問題後，自行去Google這個關鍵字．

以下為iTerms2設定．
Preferences -> Profiles -> Text -> Font -> Meslo LG S DZ for Powerline

大家主要注意是字型的位置。

下圖為原本終端機的設定，字體不是Powerline。
描述檔 -> 文字 -> 字體 -> 更改…

更改 -> 字體集：等寬字 -> XXXXX Powerline
選最後面有Powerline就可以了
