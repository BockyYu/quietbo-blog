[Mac] 安裝Jenkins，並與github連接

Saturday, December 11, 2021 2 minutes read

使用Mac M1
Jenkins 2.319.1

安裝Jenkins

到Jenkins網站下載對應的檔案。目前我是使用

brew install jenkins-lts

安裝完成後，啟動jenkins

brew services start jenkins-lts

打開瀏覽器，輸入：

http://localhost:8080

一進去jenkins會要求使用密碼解鎖，初始密碼的路徑，就在提示上的紅色字。

解鎖後，選擇要安裝的套件

接下來是安裝時間，會稍微久一點。

安裝完成後，會要求輸入建立admin帳號，這些帳密都要記住，未來登入需要使用。

成功進入jenkins！

設定 jenkins 與Github Rerository連接

  1. 新增建置作業
  2. 新增項目名稱，建置Free-Strle軟體專案
  3. 輸入github的git網址
  4. 建置觸發程序，將GitHub hook trigger for GITScm polling打勾。
  5. 建置環境選擇執行Shell，並輸入下方指令後儲存。 
echo 開始建置mydjango

注意：這次示範使用mac的shell，如果使用windows的Shell語法要注意更換。

  1. 儲存後點擊「馬上建置」，成功後左下方會出現綠色勾，若失敗的話可進入觀看失敗訊息。

建置完成後，點擊「#1」並進入「console output」，就可以看到終端機的輸出了。

Jenkins檢查github

不建議使用，基本上應該是github上有push新的程式後，jenkins再進行建置。
