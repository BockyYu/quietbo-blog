[Docker] restart

Wednesday, July 27, 2022 1 minute read

原文檔:

  • Docker Doc
  • Start containers automatically

只要重開機後, 就要手動去開啟容器, 雖然有GUI很方便按個按鈕就好, 可是有時候還是會忘記要啟動必要容器…

設置重啟策略

docker run --restart={Policy} {Container Name}

此命令更改已在運行的容器:

docker update --restart={Policy} {Container Name}

下方為Policy與原檔描述
